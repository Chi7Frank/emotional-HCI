Session 1: Hero Introduction

This project demonstrates the affective aspect of Human-Computer Interaction (HCI). The system interacts with users using text, colors, shapes, and sounds to provide feedback. Each interaction is designed to respond to user behavior in an emotional and intuitive way. The goal is to show how computers can adapt and communicate more naturally with humans.
